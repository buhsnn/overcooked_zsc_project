1. Overcooked-AI is not on pip : you must clone it and install it as a local package

First : 
git clone https://github.com/HumanCompatibleAI/overcooked_ai.git
cd overcooked_ai
Then : 
pip install -e .

2. Install Overcooked-AI dependencies

pip install numpy gymnasium pygame opencv-python tensorflow

3. Install Stable-Baselines3 (RL algorithms)

4. pip install stable-baselines3[extra]
